# VIMSUGAR

## Commands:
### Lesson 1:
```
di"
```

```
di'
```
	
Вызываем интерактивный урок в терминале
```
vimtutor
```

### Lesson 2:
Включаем подсветку строк
```
:set number
```
```
:set expandtab 
:set tabstop=2
```

```
:set hlsearch 
:set incsearch
```

Можно найти слово, если нажать на "*", находять на этом слове
